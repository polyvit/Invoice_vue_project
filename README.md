# Invoices App

_Completed in Mar, 2024_

[See demo](https://polyvit-invoices-project.netlify.app/)

Приложение инвойсов. Здесь вы можете сформировать инвойс, поменять его статус, отфильтровать по статусу, откорректировать данные.

Tools:

- HTML5, SCSS
- JavaScript
- Vue
- Vuex
- Vue-router
- Vite
- Firebase
- Uid
